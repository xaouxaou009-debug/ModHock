[About simple use]
    <Quick Modification Button> Check the box in the upper right corner of the easy-to-use button to activate the shortcut button. (There will be an additional edit button on the information of characters, objects, buildings, and plants, which is very convenient.)
    <Multiple game speed> can be turned on or off in the upper right corner of the simple use. When the extra multi-speed button (the 4th button) is lit, pressing the E button will directly reduce the game speed of the 3rd button, not slowly.
    <Auxiliary line> Auxiliary lines will appear when you use simple use to select items. Personally, I don't think it has much effect.
    <How to obtain items> Follow the prompts. When the interface reaches the upper left corner, just move the mouse to the item to be obtained, and then press the "p" key again.
    <How to obtain map coordinates> Follow the prompts. When the interface reaches the upper left corner, you only need to move the mouse to the point where you want to obtain the coordinates. The last row of the interface will display the map coordinates corresponding to the grid.
    
[About NPC]
    <Detailed attribute modification> 1. After selecting the character, use the attribute list on the right side of the character management interface. 2. After modification, click Modify List and Confirm List. (After modifying an attribute, you need to click the modify list. The confirmation list can be clicked after all the attributes are modified.) ps: Most of the desired attributes are in this list.
    <Character Management Shortcut Function> After selecting the character, click the shortcut button to realize the corresponding function.
    <Character removal> a. Easy to use to remove. b. Use the map to modify and remove. c. Use the shortcut keys in the character management to remove directly (recommended).
    <Basic requirements (elementary, fullness, etc.)> a. Use the shortcut keys in the character management (full demand, all empty demand). b. Use the demand relationship function module in character management.
    <Five Dimensions and Skill Level> is a detailed attribute modification, which is still in the list on the right side of the character management.
    <Title modification> 1. Use the Cheats button in the character management to enter the relevant interface. 2. Modify to the desired title.
    <Gongfa Modification> 1. Use the Kungfu Cheats button in the character management to enter the relevant interface. 2. Use the relevant modules to modify the exercise method you want (the input box in front is the practice level after the exercise method is modified [you can enter 1-12, corresponding to the Qi training to Yuanshen]).
    <Ordinary animal turns into a monster> After selecting npc in the character management interface (click AllNPC at the bottom left to display all npc), press the shortcut button to become a monster.
    <Rebuild Body and Heal> Use the corresponding shortcut button in character management.
    <Gongfa Cheats added> 1. Cheats for character management. 2. Add items and add exercises cheats.
    <Character Race Modification> After selecting the character, modify the "race" attribute in the attribute list on the right side of the character management interface. (Note: This attribute can be entered in the race number (check in the character call), or the English name of the race!!! When the NPC you modify is a physical training with a race-specific secret body, the archive may be damaged, so you must Remove the special secret body of this race before modification. For example: chicken -> rabbit, the comb and feathers need to be removed)

[About the type of attributes in character management]
    <Type 1> Additional value v, addv. It is a real added value and does not receive other bonus effects. (Type 1 is generally sufficient.)
    <Type 2> Additional value p, addp. The added value affected only by its own ratio. (ps: The character skill level can be changed to Type 2 first, and then modified to break the upper limit.)
    <Type 3> Basic value v, basev. Will be affected by bonuses. The bonus may be the bonus of Gongfa and so on.
    <Type 4> Basic value p, basep. Will be affected by the bonus, but also by its own ratio.
